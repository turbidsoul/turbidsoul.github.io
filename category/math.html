之前买了几本关于数学的书，这两天看了《程序员数学》第一章 0的故事，以前只是知道0，却不知道0在数学中的作用和意义。

计数法

在10进制中，每一位的数字表示的是这一位上的个数，如书中的2503：

2 => $2 xx 1000$ => $2 xx 10^3$

5 => $5 xx 100$ => $5 xx 10^5$

0 => $0 xx 10$ => $0 xx 10^1$

3 => $3 xx 1$ => $3 xx 10^0$
